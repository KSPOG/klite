# KLite GE Flipper

KLite marketplace source for the Flipping Copilot automation helper.

Current version: **0.3.13**  
Minimum client: **1.0.98**

Version 0.3.13 dispatches known GE setup actions directly, uses KLite Core canvas input for visual fallbacks, and backs out after three unchanged Copilot-highlight attempts instead of clicking indefinitely.
